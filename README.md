# 🧸 Maven Toys Sales Analysis | SQL + Power BI

## 📌 Project Overview

This project demonstrates an end-to-end sales analysis workflow using **SQL Server** for data analysis and **Microsoft Power BI** for interactive 
dashboard development.

The objective was to analyze the sales performance of Maven Toys by answering key business questions related to revenue, profit, product performance
and store performance. Insights generated through SQL analysis were then transformed into interactive Power BI dashboards for executive reporting.

# 💡 Key Business Insights

- Toys generated the highest revenue among all product categories.
- Lego Bricks generated the highest overall revenue.
- Colorbuds generated the highest overall profit.
- Ciudad de Mexico 2 generated the highest store revenue.
- Top-performing products contributed significantly to total business revenue.
